OSWistia
============

## About

With this plugin, you can embed Wistia videos into articles simply by adding a tag like {wistia}thevideoid{/wistia} into an article with fluid responsive support.

## Params

* width
* height
* autoplay
* focus
* captions
* nextURL
* previousURL
* homeURL
* showAutoplayControl
* showFocusControl
* showDownloadControl
* showNavigationControl

## Requirements

Joomla 2.5.x or 3.x
